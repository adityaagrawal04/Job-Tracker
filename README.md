# Job Tracker

A modern, efficient web application designed to help you organize and track your job search process. This project was created as part of the **Build and Blog Workshop**.

Built with **React**, **TypeScript**, and **Vite**, this application provides a streamlined interface to manage job applications, interview statuses, and company details.

## ğŸ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Make sure you have the following installed:

  - [Node.js](https://nodejs.org/) (v14 or higher recommended)
  - [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/adityaagrawal04/Job-Tracker.git](https://github.com/adityaagrawal04/Job-Tracker.git)
    cd Job-Tracker
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or if you use yarn
    yarn install
    ```

### Running the App

Start the development server:

```bash
npm run dev
# or
yarn dev
```

Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal) to view the app.

### Building for Production

To build the app for production deployment:

```bash
npm run build
```
